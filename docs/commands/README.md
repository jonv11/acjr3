# Commands

## Core commands

- `acjr3 request <METHOD> <PATH> [options]`: universal Jira REST proxy
- `acjr3 config ...`: configuration helpers (`check`, `show`, `set`, `init`)
- `acjr3 openapi ...`: fetch and inspect OpenAPI metadata

## Jira shortcut commands

- `acjr3 issue ...`
